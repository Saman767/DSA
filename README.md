# DSA

A personal collection of Data Structures and Algorithms solutions in C++, organized by difficulty.

## Structure

```
.
├── Easy/
│   ├── Remove_Duplicates_From_Sorted_List.cpp
│   ├── Two_Sum_II.cpp

│   ├── Add solution for Remove Duplicates from Sorted Array
│   └── sliding_window/
│       ├── min_length.cpp
│       └── min_sum.cpp
├── Medium/
│   ├── 3Sum.cpp
│   ├── Dutch National Flag problem

│   ├── triplet sum to zero

    └── sliding window/
```

## Topics Covered

- **Arrays & Two Pointers** — Two Sum II, Sorted Array Squares, Remove Duplicates, 3Sum, Triplet problems, Dutch National Flag
- **Sliding Window** — Min subarray length, Max subarray sum, Fruits into Basket, Longest substring with K distinct characters, Longest substring without repeating characters, Longest substring with same letter after replacement
- **Linked Lists** — Remove Duplicates from Sorted List

## Usage

Each file contains a standalone C++ `Solution` class in the style used on LeetCode / GeeksforGeeks. To compile and test a specific solution locally, wrap it in a `main` function and build with:

```bash
g++ -std=c++17 -O2 -o solution path/to/file.cpp
./solution
```
